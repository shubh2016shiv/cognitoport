# Shubham Singh — Generative AI Portfolio

---

## 日本語版

### 概要

機械学習エンジニアであるShubham Singhのポートフォリオサイト。医療分野の生成AI、本番環境レベルのLLMシステム、およびエージェント型RAGパイプラインを専門としています。

バニラJavaScript、Tailwind CSS、Canvas APIを使用して構築されており、フレームワークやバンドラーは使用していません。

### 主な機能

- **CLIターミナルヒーロー**: タイプライターアニメーションによるインタラクティブなイントロダクション
- **プロフィール**: 医療AI、RAGパイプライン、機械学習エンジニアリングの経験を紹介
- **プロジェクト紹介**: 主要なAIプロジェクトを水平スクロールカルーセルで表示
- **技術スキル**: スクロール連動アニメーションで4つのスキルカテゴリを表示
  - 生成AI
  - 機械学習
  - 自然言語処理
  - クラウド & MLOps
- **キャリアタイムライン**: 展開可能なインタラクティブな職歴表示
- **お問い合わせフォーム**: FormSubmit.co経由での機能的なコンタクトフォーム
- **履歴書ダウンロード**: GitHubリポジトリから最新のPDFを動的に取得

### 技術スタック

| 技術 | 用途 |
|---|---|
| HTML5 | ページ構造 |
| CSS3 | カスタムスタイル、アニメーション |
| Tailwind CSS (CDN) | ユーティリティファーストスタイリング |
| Vanilla JavaScript | インタラクティビティとロジック |
| Canvas API | パーティクルアニメーション、可視化 |
| FormSubmit.co | サーバーレスフォーム送信 |

### ローカル実行

```sh
npx live-server
```

ブラウザで `http://localhost:8000` を開きます。

### プロジェクト構成

```

├── index.html              # メインHTMLページ

├── skills.js               # スキルアニメーションエンジン
├── contact.js              # お問い合わせフォーム
```

## English Version

### Overview

Portfolio website for **Shubham Singh**, a Machine Learning Engineer specializing in healthcare generative AI, production-grade LLM systems, and agentic RAG pipelines.

Built with vanilla JavaScript, Tailwind CSS, and Canvas API — no frameworks or bundlers required.

### Key Features

- **CLI Terminal Hero**: Interactive typewriter animation introduction
- **About Me**: Professional narrative highlighting healthcare AI, RAG pipelines, and ML engineering experience
- **Featured Projects**: Horizontally scrollable carousel showcasing key AI projects
- **Technical Arsenal**: Scroll-triggered animation displaying four skill categories:
  - Generative AI
  - Machine Learning
  - Natural Language Processing
  - Cloud & MLOps
- **Career Journey**: Interactive timeline with expandable role details
- **Contact Form**: Functional contact form via FormSubmit.co
- **Resume Download**: Dynamically fetches latest PDF from GitHub repository

### Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Custom styles and animations |
| Tailwind CSS (CDN) | Utility-first styling |
| Vanilla JavaScript | All interactivity and logic |
| Canvas API | Particle animations and visualizations |
| FormSubmit.co | Serverless form submissions |

### Local Development

```sh
npx live-server
```

Open `http://localhost:8000` in your browser.

### Project Structure

```
potfolio/

├── career_journey.js       # Career timeline data
├── skills.js               # Skills animation engine
├── contact.js              # Contact form logic
└── assets/
    └── resume/             # Resume PDF files
```
